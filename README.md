## Run Locally

Clone the project

```bash
  git clone https://github.com/Emin14/to-do-list.git
```

Go to the project directory

```bash
  cd to-do-list
```

Start 

```bash
  open index.html
```


## Features

- Adding new categories
- Adding new tasks
- Editing a task
- Deleting a task
- Marking a task as completed
